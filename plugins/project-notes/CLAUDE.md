# project-notes

Project knowledge lives in `notes/` at the repository root.

Three skills handle different types of knowledge:

- **project-notes** -- wiki pages and general documentation in `notes/`
- **decision-log** -- structured decision records in `notes/DECISIONS.md`
- **journal** -- timestamped observations in `notes/journal/`

Use `[[wikilinks]]` for internal cross-references between notes.
